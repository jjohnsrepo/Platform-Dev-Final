# Getting Started

## Prerequisites

- [Node.js](https://nodejs.org/) installed on your computer
- [Expo Go](https://expo.dev/client) downloaded on your phone
  > ⚠️ **Note:** This app has only been tested on iOS. Android results may vary.

## Setup Instructions

1. **Clone the repository** onto your local computer
2. **Open a terminal** in the project folder
3. **Install dependencies:**
   ```bash
   npm install
   ```
4. **Start the development server:**
   ```bash
   npx expo start
   ```
5. **Open the app** by scanning the QR code displayed in the terminal using the Expo Go app on your phone
